# UnderDream

### 游戏引擎：
>> u3d

### 开发工具：
>> Unity、Visual Studio 2019、Visual Studio Code

### 界面UI：
>> 2D、正45°仰角、像素

### 游戏性：
>> 剧情向、RPG

### 概括剧情：
>> 大型企业员工及相关人员集体穿越到异世界后，深化矛盾的杀人吃鸡游戏

### 剧情推动：
>> 多事件循环触发推动剧情、手动选择触发事件、系统设定触发特定事件

### 判定机制：
>> 类DnD骰子机制、多标签判定机制、属性检测机制

### 游戏目标：
>> 异世界不存在对女主含有敌意的人，且女主存活

### 死亡惩罚：
>> 多属性减益、标签更新、重回异世界存档点

### 失败条件：
>> 女主进入死亡判定

### 剧情分支：
>> 多分支，各人物由若干数值综合判断触发剧情

### 游戏结局：
>> GoodEnding(男女主存活且若干数值判定通过)、NormalEnding(其他)、BadEnding(女主死亡)

### 游戏分级
>> R18
